# Instituto Nacional - API REST

Este proyecto es una API REST desarrollada para la gestión del **Instituto Nacional**. Permite administrar la información de estudiantes, roles y autenticación de usuarios de manera segura.

## 🚀 Tecnologías Utilizadas

*   **Node.js** (Entorno de ejecución)
*   **Express** (Framework web para las rutas y middlewares)
*   **Prisma ORM** (Mapeador de base de datos y migraciones)
*   **JSON Web Tokens (JWT)** (Para la autenticación y protección de rutas)
*   **Swagger** (Para la documentación de la API)
